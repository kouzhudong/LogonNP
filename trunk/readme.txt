目录和文件说明：
1.LogonNP
  SDK的名字：Windows® Server 2003 SP1 Platform SDK ISO。谨记：以后的SDK都没有这个工程了。
  下载路径：https://download.microsoft.com/download/7/5/e/75ec7f04-4c8c-4f38-b582-966e76602643/5.2.3790.1830.15.PlatformSDK_Svr2003SP1_rtm.img
  工程位置：C:\Program Files\Microsoft Platform SDK\Samples\Security\NetProviders\LogonNP\LogonNP.c
2.LogonNP-2019
  这个是VS2019的命令行可编译的工程配置。
  打开:x64 Native Tools Command Prompt for VS 2019或者x86 Native Tools Command Prompt for VS 2019。
  切换到这个目录。
  直接输入：nmake，然后回车即可编译。
3.LogonNP-vs2019
  自己弄的可直接用vs2019编译的工程。
4.register.bat
  注册的脚本。
  最后一个设置需要手动完成。
5.delete.bat
  反注册的脚本。
  最后一个设置需要手动完成。
